# 🔐 OPEN Password Generator

A modern and secure password generator built with JavaScript.
Generate strong, customizable passwords instantly with a clean and interactive UI.

---

## 🚀 Live Demo

👉 https://premiumpasswordgenerator.web.app/

---

## ✨ Features

* Generate secure random passwords
* Customize password length and character types
* Copy to clipboard with instant feedback
* Password strength indicator (weak / medium / strong)
* Save password history
* Dark / Light mode toggle 🌙
* Fully responsive design (mobile & desktop)

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---

## ⚙️ How to Run Locally

```bash
git clone https://github.com/Hussein35847/PasswordGeneratorWebApp
cd open-password-generator
open index.html
```
